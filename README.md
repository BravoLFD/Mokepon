# Project Mokepon
Multiplayer game developed as part of Platzi’s free Basic Programming course, enhanced with custom logic. This project helped me gain essential backend and frontend skills using JavaScript, HTML, CSS, and Node.js.

# About this project
This project was developed based on Platzi’s free Basic Programming course, incorporating some adjustments and improvements based on my own logic. The game includes multiple separate JavaScript files: one handles the multiplayer functionality, while another manages a single-player mode featuring combat against the computer. Through this process, I gained fundamental knowledge in both backend and frontend, using technologies such as JavaScript, HTML, and CSS, as well as working with Node.js for the server development in the multiplayer version of the game.

# Features

Multiplayer functionality with real-time interaction
Single-player mode with combat against the computer
Separate JavaScript files for multiplayer and single-player versions
Frontend built with JavaScript, HTML, and CSS
Backend server implemented using Node.js
Custom improvements in game logic beyond the original course material

Although the game works correctly, there are still several improvements to be made to optimize the experience and add new features.
